Node cli tool for scraping chilean postal codes

## Requirements

- node.js >=4

## Install

```bash
npm install
```

## Running

For cli prompt:
```bash
node find-postal-code.js
```

Or you can pass the arguments as flags:

```bash
node find-postal-code.js --street waterloo --street_number 1522 --commune "las condes"
```
